Make `DOM Manipulation`_ in Go_ as similar to JavaScript_ as possible via
WebAssembly_. For DOM Manipulation via GopherJS_, visit root_ directory.

  - `Ubuntu 18.04`_
  - Go_ (1.11 or later)


Install
+++++++

.. code-block:: bash

  $ GOARCH=wasm GOOS=js go get -u github.com/siongui/godom/wasm
