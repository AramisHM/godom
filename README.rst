==========================
`DOM Manipulation`_ in Go_
==========================

Make `DOM Manipulation`_ in Go_ as similar to JavaScript_ as possible.
(via GopherJS_)

  - `Ubuntu 16.10`_
  - `Go 1.7.5`_
